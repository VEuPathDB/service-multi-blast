Constructs a new SequenceFileWriter instance.

Parameters

fileNamePrefix

Prefix for the names of the files generated by the execution of this file writer.

File names will follow the pattern ${fileNamePrefix}-${fileIndex}.txt.

${fileIndex} will be an incrementing int value starting from 0 for the overall query file, and continuing for individual sequences in a multi-sequence query.

For example, given the fileNamePrefix value "foobar" a query with a single sequence would generate the file name:

foobar-0.txt
