Consumes the given input stream and pipes the sequences out to a collection of files.

The returned result will contain an overall query file which contains the entirety of the input stream contents.

The returned result will also contain a list of sub-query files for each individual sequence if and only if the input query contained more than one sequence.

If the input stream contains only one sequence, the returned result sub-query list will be empty as the overall query file is the only result.
