Protein Sequence Validation Stream

SequenceValidationStream implementation that validates the streamed query sequences against the following rules:

  1. All characters in the query sequences are in the legal character set for protein sequences.

  2. No sequences in the query contain more than the configured max number of non-ignorable characters.

  3. The overall query contains no more than the configured max number of sequences.

  4. The overall query contains no more than the configured max number of non-ignorable characters.
